**找出从名为access.log日志文件访问数量最大的IP地址和访问的次数**

```
tail -100000 access.log | awk '{print $1}' | sort | uniq -c | sort -nr 
```

**找出最常用的10条命令**

```
history | awk "{print $2}" | sort | uniq -c | sort -n -r -k 1 | head -n 10
```
